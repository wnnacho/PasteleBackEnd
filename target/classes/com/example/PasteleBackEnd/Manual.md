# Manual de Usuario - Pastelería Mil Sabores

## 1. Acceso al Sistema

### Usuario Público:
1. Navegar a `http://localhost:3000`
2. Explorar productos sin login
3. Agregar productos al carrito
4. Comprar como invitado

### Administrador:
1. Click en "Inicio de sesión"
2. Ingresar credenciales:
   - Email: `admin@pasteleriamilsabores.cl`
   - Contraseña: `admin123`
3. Acceder a panel admin en `/admin`

## 2. Funcionalidades Principales

### Catálogo de Productos:
- Ver todos los productos
- Filtrar por categoría
- Ver detalles con modal
- Agregar al carrito

### Carrito de Compras:
- Agregar/eliminar productos
- Modificar cantidades
- Personalizar productos
- Calcular total
- Procesar compra

### Panel de Administración:
- **Dashboard**: Estadísticas
- **Productos**: CRUD completo
- **Pedidos**: Gestión de compras
- **Usuarios**: Gestión de clientes
- **Blogs**: Gestión de contenido
